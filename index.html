Meu foco é transformar dados em ativos estratégicos para o negócio, desenvolvendo soluções completas de Business Intelligence, desde a preparação dos dados até a construção de indicadores executivos para apoio à tomada de decisão.

Ao longo da minha trajetória participei de projetos envolvendo implantação SAP S/4HANA, plataformas analíticas, integração de APIs, Data Lake, Power BI, SQL, Python e Apache Airflow, sempre buscando gerar valor por meio dos dados.

Casos de Business Intelligence

Coleção de soluções analíticas desenvolvidas em ambiente corporativo para apoiar decisões estratégicas, automatizar processos e transformar dados em valor para o negócio.

    Power BI • Python • SAP Cloud ALM • DAX

  • Governança Executiva da Implantação SAP

    Power BI • SAP • Analytics

    SQL • Python • Airflow • Power BI

    Solução analítica para identificar oportunidades de redução de custos relacionados ao licenciamento ERP, utilizando integração de dados, processamento automatizado e dashboards executivos.

    Power BI • SQL • Salesforce

    Monitoramento executivo do retorno sobre investimentos, forecast financeiro, custos, receitas e indicadores estratégicos da operação comercial.
